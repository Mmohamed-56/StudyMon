# Sound Effects

Place static MP3 sound effect files in this directory.

## Required Sound Files:

### UI Sounds
- `button_click.mp3` - Button press
- `tab_switch.mp3` - Tab navigation

### Battle Sounds
- `attack_hit.mp3` - Normal attack
- `super_effective.mp3` - Super effective hit
- `not_effective.mp3` - Not very effective

### Creature Sounds
- `creature_switch.mp3` - Switching creatures
- `creature_faint.mp3` - Creature faints
- `creature_heal.mp3` - Healing
- `level_up.mp3` - Level up

### Question Sounds
- `correct_answer.mp3` - Correct answer
- `wrong_answer.mp3` - Wrong answer
- `sp_gain.mp3` - SP gained

### Victory Sounds
- `badge_earned.mp3` - Badge earned
- `gym_victory.mp3` - Gym victory
- `catch_success.mp3` - Caught creature

## Generation Instructions:

You can use ElevenLabs Sound Effects to generate these once, then download and place here.

Visit: https://elevenlabs.io/sound-effects

Descriptions in: `src/data/soundEffects.js`
